#### Property Value
[ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.htm) InvalidElementId signals that there is no permanent preview view for the document. Where possible the document's active view will be used to generate the preview upon save; where there is no active view the document will not include a preview. 
Exception| Condition  
---|---  
[ArgumentException](2e6e4206-97a8-dd4b-df5d-4269f4bb6088.htm)|  When setting this property: The view id is not valid to use as a preview view.   
[ArgumentNullException](631e1424-60f4-929b-4e52-dda9dcd26316.htm)|  When setting this property: A non-optional argument was null   
This view will always be saved as the preview view for the document unless it is overridden by the value used in SaveOptions or SaveAsOptions.
