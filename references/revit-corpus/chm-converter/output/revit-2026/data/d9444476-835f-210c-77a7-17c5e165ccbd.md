# RevitLinkGraphicsSettings.ViewRange Property

---  
The view range of RevitLinkGraphicsSettings. The view range can be set to [ByHostView](f33e82f3-5bec-4532-457e-d01dc537024f.htm) or [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm). Only a view that supports view range can have this property set to [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm). To check if the view supports the view range use [!:Autodesk::Revit::DB::LinkVisibility::IsViewRangeSupported(Autodesk::Revit::DB::LinkVisibility)]. The [LinkedViewId](053b6a8c-2212-322c-8f21-b7d95e089b42.htm) parameter must have a value assigned if this property is set to [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm).
