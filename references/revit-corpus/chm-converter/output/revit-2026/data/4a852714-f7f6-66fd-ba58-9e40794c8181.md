# RevitLinkGraphicsSettings.ViewFilterType Property

---  
The view filter type of RevitLinkGraphicsSettings. If the [LinkVisibilityType](e48416ed-38f3-063d-cddd-8262486772b6.htm) is set to [ByHostView](f33e82f3-5bec-4532-457e-d01dc537024f.htm), then the view filter type should be set to [ByHostView](f33e82f3-5bec-4532-457e-d01dc537024f.htm). If the [LinkVisibilityType](e48416ed-38f3-063d-cddd-8262486772b6.htm) is set to [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm), then the view filter type should be set to [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm). If the [LinkVisibilityType](e48416ed-38f3-063d-cddd-8262486772b6.htm) is set to [Custom](f33e82f3-5bec-4532-457e-d01dc537024f.htm), then the view filter type can be set to any value from [LinkVisibility](f33e82f3-5bec-4532-457e-d01dc537024f.htm) enum. The [LinkedViewId](053b6a8c-2212-322c-8f21-b7d95e089b42.htm) parameter must have a value assigned if this property is set to [ByLinkView](f33e82f3-5bec-4532-457e-d01dc537024f.htm). The [Custom](f33e82f3-5bec-4532-457e-d01dc537024f.htm) value corresponds to None option in the UI.
