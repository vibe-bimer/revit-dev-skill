#### Parameters
host [Element](eb16114f-69ea-f4de-0d0d-f7388b105a16.htm)
    Host elements that new opening would lie in. The host can only be a wall or a ceiling.
profile [CurveArray](55103aad-38fd-45d2-6bf7-67a5203e99f3.htm)
    The profile of the newly created opening. This may contain more than one curve loop. Each loop must be a fully closed curve loop and the loops may not intersect. The profiles will be projected into the host plane.
#### Return Value
[Opening](720de864-9f4e-c606-c7f4-2e7a0b2da46f.htm)If successful, the newly created opening is returned, otherwise an exception with error information will be thrown.
