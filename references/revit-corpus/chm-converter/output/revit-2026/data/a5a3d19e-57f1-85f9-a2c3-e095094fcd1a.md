# MultiServerService.SetServerState Method

#### Parameters
serverId Guid
     Id of the server to switch active state for. 
document [Document](db03274b-a107-aa32-9034-f3e0df4bb1ec.htm)
     Document for which to activate this server. If null, server will activate globally. 
bActive Boolean
     True to activate server, false to deactivate. 
#### Return Value
Boolean True if operation succeeded (even if the server state was not changed), false otherwise (e.g. when serverId is invalid). 
Exception| Condition  
---|---  
[InvalidOperationException](9e715f03-3884-e539-4dd6-8d7545733adc.htm)|  For a service that doesn't support activation, the servers can't be activated/deactivated.   
For a service which has cref="Autodesk::Revit::DB::ExternalService::SupportActivation" set to false calling this method will throw exception.
