#### Property Value
String
The path string can be in one of several formats: absolute local, relative local, or server (indicating that the file was provided by an external server). The format is determined by [PathType](c50808f3-77e5-9606-ec79-721c03873ed2.htm)
The path can be an empty string, for example for (but not limited to) ImageTypes created using "Save to Project as Image".
