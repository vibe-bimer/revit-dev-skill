#### Property Value
String The required format for this string is a collection of one or more filter labels (representing the text the user will see in the Files of Type pulldown) plus an associated file matching string separated by the '|' character. Some examples of valid filter strings: 
  * "Revit Files (*.rvt)|*.rvt
  * "Revit Projects (*.rvt)|*.rvt|Revit Families (*.rfa)|*.rfa"
  * "All files|*.*"
  * "All Revit files (*.rvt, *.rfa, *.rte, *.rft)|*.rvt;*.rfa;*.rte;*.rft"
