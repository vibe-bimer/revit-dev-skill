[ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.htm) The id must be valid for use as a preview view (check DocumentPreviewSettings.IsViewIdValidForPreview()). Revit does not check validity when setting this value, only when the options are used in a call to SaveAs().
