#### Property Value
[Room](75c9d2c7-a402-ea8b-9e7c-f8bc3510bbd5.htm)The "To Room" set for the door or window in the last phase of the project.  if the FamilyInstance is neither a door nor a window, or if the door or window leads to a space where there is no room (an exterior space or interior space with no room defined). Also  if there is no target room assigned to the last phase, or phase does not apply to this FamilyInstance.
Revit automatically calculates the "To Room" for a given door or window. This is the default shown when creating a door or window schedule and adding the "To Room" properties. The user can opt to swap the "From Room" and "To Room" values via the schedule if they choose. The method [FlipFromToRoom](ae1158c1-1fb0-0558-0ea4-e1cf76bb8a1e.htm) also can be used to swap the values.
