This event is only supported for Collaboration for Revit and will not be raised for those operations occurring in local-worksharing and Revit Server workflow. Users may not change the document in the handler for this event. If you do not need detailed progress information for synchronizing with central, see [!:Autodesk::Revit::DB::Events::DocumentSynchronizingWithCentral] or [!:Autodesk::Revit::DB::Events::DocumentSynchronizedWithCentral]. It is NOT recommended to do any time-consuming work when handling DocumentSynchronizingWithCentralProgessChanged event. This can increase workshared operation time." Exception [ModificationForbiddenException](53205486-5917-7c33-8e67-e362106ddc97.htm) will be thrown if any document-modifying method is called during this event's handler.
