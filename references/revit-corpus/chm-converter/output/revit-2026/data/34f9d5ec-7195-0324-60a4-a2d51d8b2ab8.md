# EnergyAnalysisSpace.GetBoundary Method

---  
Gets the collection of polygons that form the 2D boundary. This method returns a collection of polyloops (planar polygons) that defines an enclosed area measured by interior bounding surfaces, if in rooms and spaces mode. In other modes, returns no polyloops.
