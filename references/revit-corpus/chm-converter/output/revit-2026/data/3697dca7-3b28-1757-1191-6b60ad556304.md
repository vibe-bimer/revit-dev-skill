#### Property Value
Int32
The effective priority value should be between 1-5. Membrane layers should not have a priority, the value should be 999. You should set the layer priority if you want a customized priority value rather than the default one by function. In shell, a non-membrane layer which is closer to the core should not have a lower priority.
