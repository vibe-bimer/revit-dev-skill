#### Return Value
[ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.htm) The id of the immediate parent of this link, or invalidElementId if this link is a top-level link. 
This function returns the immediate parent id. Given the link structure A -> B -> C, then calling this function on C will return B's id. Call GetRootId to get A's id.
