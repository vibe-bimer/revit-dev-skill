IList[ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.htm) Returns ids of elements which belong to that sketch. 
Following elements can belong to a sketch: [ModelCurve](f15a85c2-3aee-9055-f9f8-9001b47fcefb.htm), [ReferencePlane](e7003ec7-1dbe-50a2-fb3d-a83a5a3b5b9f.htm), [Dimension](210f88be-e3c5-26a4-7dd8-3296f6725cce.htm).
