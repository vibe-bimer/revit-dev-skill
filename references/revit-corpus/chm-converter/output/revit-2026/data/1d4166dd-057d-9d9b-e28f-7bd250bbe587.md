#### Property Value
Boolean
A family is parametric if it contains parameter-driven relations between its elements, such as labeled dimensions or locked tangential joins.
Parametric families containing large sketches may cause performance problems. Caution is therefore advised before adding any constraints to a yet non-parametric family that already contains large sketches. See [HasLargeSketches](c49820d6-d8f2-de9a-544d-f896a32e39c8.htm).
