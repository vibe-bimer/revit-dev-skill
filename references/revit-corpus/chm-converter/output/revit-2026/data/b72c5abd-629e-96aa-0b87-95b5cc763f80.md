# BRepBuilder.GetResult(ExternalGeometryId, BRepBuilderPersistentIds) Method

---  
Get the [ExternallyTaggedBRep](58a208f6-2ce5-d6cf-d17e-f4968fae5b31.htm) built by this BRepBuilder. This will clear the built geometry stored in the BRepBuilder. This function will throw if this BRepBuilder hasn't completed building the b-rep. Use IsResultAvailable() to verify that this BRepBuilder contains a valid result.
