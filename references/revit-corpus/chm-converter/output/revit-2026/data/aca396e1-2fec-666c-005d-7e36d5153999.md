#### Property Value
Double The row height for any body rows in the schedule. 
Exception| Condition  
---|---  
[ArgumentOutOfRangeException](60f148c9-ece0-a6bb-4e12-bb4a9c8c8a24.htm)|  When setting this property: The given value for rowHeight must be between 0 and 30000 feet.   
[InvalidOperationException](9e715f03-3884-e539-4dd6-8d7545733adc.htm)|  The RowHeightOverride property is set to RowHeightOverrideOptions.None.   
Setting this property will force all or image only (depending on how [RowHeightOverride](92cfddb5-13af-dc32-0b27-2f0fc35dbae2.htm) is defined) schedule body rows to be resized to the indicated height value (when viewed as a ScheduleSheetInstance on a ViewSheet). If the value that is set is smaller than the height of the text (single or multi line), then that value is not taken into account. To access or set this property the [RowHeightOverride](92cfddb5-13af-dc32-0b27-2f0fc35dbae2.htm) property should not be [None](3f75752e-c4df-90c9-e296-ac604f8c4fd9.htm)
