If the input linkId references [RevitLinkType](2204a5ab-6476-df41-116d-23dbe3cb5407.htm), then the link overrides will be set to default. If the input linkId references [RevitLinkInstance](a3a27c39-75bf-67d1-ae78-4cadd49a9c8e.htm), then the link overrides will be removed and the settings of [RevitLinkType](2204a5ab-6476-df41-116d-23dbe3cb5407.htm) will be used for this instance.
