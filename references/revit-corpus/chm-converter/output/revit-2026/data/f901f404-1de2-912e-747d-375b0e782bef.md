#### Return Value
Boolean True if the sub-transaction has already started, False otherwise. 
A sub-transaction is considered being started after the [Start](24a1e46d-2893-231a-cfff-a3f8e411247a.htm) method was called and until the sub-transaction is either committed or rolled back.
