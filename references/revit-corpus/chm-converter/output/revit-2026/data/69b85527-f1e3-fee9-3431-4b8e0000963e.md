#### Parameters
dBView [View](fb92a4e7-f3a7-ef14-e631-342179b18de9.htm)
     The view where rendering will occur. If this argument is , a view-independent bounding box should be reported. 
#### Return Value
[Outline](1ffe9215-0dd5-358f-495d-e983f9e7d295.htm) The bounding box as an Outline. 
Revit uses the bounding box when navigating views, e.g., when a Zoom to Fit command is issued. The reported bounding box does not have to be tight. However, there may be unintended side-effects if the box is inconsistent with the submitted geometry.
