#### Parameters
layerIdx Int32
     Index of a layer in the CompoundStructure. The layer index is zero based. It counts from the exterior of wall and from the top of roofs, floors and ceilings. 
#### Return Value
Boolean Returns true if the layer is within the core layer boundary, false if it is in the interior or exterior shell layers. 
You can change the shell/core layer boundary using [SetNumberOfShellLayers(ShellLayerType, Int32)](8b8ea4e3-e697-6176-92c0-dc2687723a71.htm).
