# BRepBuilder.GetResult Method

---  
Get the [Solid](7a3b5ac1-c66d-9f81-a11d-9bcd4e026295.htm) built by this BRepBuilder. This will clear the built geometry stored in the BRepBuilder. This function will throw if this BRepBuilder hasn't completed building the b-rep. Use IsResultAvailable() to verify that this BRepBuilder contains a valid result.
