# RouteAnalysisSettings.EnableIgnoredCategoryIds Property

---  
When this setting is true, elements with category ids returned by [GetIgnoredCategoryIds](e4115866-1e15-9b46-d783-7fc5786e88c5.htm) will be ignored in route calculation. default is true as all elements with the Door Category Id are ignored by default.
