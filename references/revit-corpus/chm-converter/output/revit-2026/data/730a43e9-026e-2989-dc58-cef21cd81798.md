#### Return Value
Boolean True if the transaction group has already started, False otherwise. 
A transaction group is considered being started after the [Start](fff3e88e-358c-e6d0-d539-61517f53140c.htm) method was called and until the transaction group is either completely committed or rolled back. A transaction group can only be Started when HasStarted returns false.
